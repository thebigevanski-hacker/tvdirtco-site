# Temecula Valley Dirt Co. Website

Static one-page site for excavation, hauling, land clearing, and equipment rentals in Temecula Valley, San Diego, Riverside.

## Tech
- HTML + Tailwind CSS (CDN)
- Vanilla JS for particles & lightbox
- Deployed on Vercel

## Local Dev
Just open `index.html` in browser.
